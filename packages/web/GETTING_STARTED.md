# Getting Started with @unillm-ts/web

本指南将帮助你快速上手 UniLLM Web 管理界面。

## 前置要求

- Node.js >= 16.0.0
- pnpm >= 8.0.0
- 运行中的 @unillm-ts/server (API 服务器)

## 快速开始

### 1. 安装依赖

从项目根目录运行：

```bash
pnpm install
```

### 2. 启动 API 服务器

在一个终端窗口中：

```bash
pnpm dev:server
```

服务器将在 `http://localhost:3000` 上运行。

### 3. 启动 Web 界面

在另一个终端窗口中：

```bash
pnpm dev:web
```

Web 界面将在 `http://localhost:5173` 上运行，并自动在浏览器中打开。

## 使用说明

### 配置管理页面

1. **查看实例列表**
   - 页面会显示所有配置实例
   - 可以看到实例名称、模板、选中的模型等信息
   - 当前激活的实例会有"当前"标签

2. **创建新实例**
   - 点击"创建实例"按钮
   - 选择配置模板（如 OpenAI、Qwen 等）
   - 输入实例名称
   - 选择要使用的模型
   - 配置密钥（如 API Key）
   - 可选：配置高级选项
   - 点击"确定"创建

3. **编辑实例**
   - 点击实例行的"编辑"按钮
   - 修改实例名称、选择的模型或其他配置
   - 密钥字段留空表示不修改
   - 点击"确定"保存

4. **删除实例**
   - 点击实例行的"删除"按钮
   - 确认删除操作
   - 注意：默认实例不能被删除

5. **切换当前实例**
   - 点击"设为当前"按钮
   - 该实例将成为聊天页面的默认实例

### 聊天页面

1. **选择实例**
   - 使用顶部的下拉菜单选择要使用的实例
   - 切换实例会自动更新系统的当前实例

2. **发送消息**
   - 在底部输入框输入你的消息
   - 按 Enter 键发送（Shift+Enter 换行）
   - 或点击"发送"按钮

3. **查看响应**
   - AI 的响应会实时流式显示
   - 支持多轮对话
   - 对话历史会保留在页面上

4. **清空对话**
   - 点击"清空对话"按钮可以清除所有消息
   - 开始新的对话

## 配置模板说明

系统预置了以下配置模板：

- **OpenAI**: 使用 GPT-4o 等 OpenAI 模型
- **Qwen**: 使用阿里云 Qwen Plus 模型
- **ZhiPu**: 使用智谱 GLM-4 模型
- **Moonshot**: 使用 Moonshot AI 的 Kimi 模型
- **Spark**: 使用讯飞星火大模型

每个模板需要相应的 API 密钥。

## 故障排除

### 无法连接到 API 服务器

确保：
1. API 服务器正在运行（`pnpm dev:server`）
2. 服务器运行在 `http://localhost:3000`
3. 没有防火墙阻止连接

### 实例创建失败

确保：
1. 已正确输入所有必填字段
2. API 密钥是有效的
3. 选择的模型在该模板中可用

### 聊天无响应

确保：
1. 已选择一个实例
2. 实例的 API 密钥配置正确
3. 有网络连接到相应的 AI 服务

## 生产环境部署

### 构建

```bash
pnpm build:web
```

构建产物将生成在 `packages/web/dist` 目录中。

### 部署

可以使用任何静态文件服务器部署构建产物，例如：

- Nginx
- Apache
- Vercel
- Netlify

确保配置正确的 API 代理，将 `/api` 路径代理到你的 API 服务器。
